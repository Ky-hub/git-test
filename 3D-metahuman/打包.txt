echo $DISPLAY


在 Project Settings → Platforms → Linux：

1. Target Configuration

Development：调试用，可打印日志，调试帧抓取

Shipping：正式版本，优化性能

2. Build Configuration

勾选 For Distribution（可选，Shipping 配合）

3. Build Options

保证 Include Debug Files 和 Symbols 根据需求选

4. Rendering

如果需要硬件加速，确保 GPU 驱动正确

如果是无头服务器，需要使用 Xvfb 或虚拟帧缓冲

  ./RunUAT.sh BuildCookRun \
    -project=/path/to/MyProject/MyProject.uproject \
    -noP4 \
    -platform=Linux \
    -clientconfig=Development \
    -serverconfig=Development \
    -cook \
    -allmaps \
    -build \
    -stage \
    -pak \
    -archive \
    -archivedirectory=/path/to/output



直接运行：
./MyProject
会打开游戏窗口（如果有 GUI 和显示器）。
如果在无显示器的服务器上，需要虚拟帧缓冲（Xvfb）：
Xvfb :0 -screen 0 1280x720x24 &
export DISPLAY=:0
./MyProject



如果你想 后台运行 并记录日志：
nohup ./MyProject > game.log 2>&1 &
查看日志：
tail -f game.log
